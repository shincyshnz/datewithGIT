git init (to create git) git add index.html (to add and then commit) git commit -m "msg for the new version" (commit the latest modification) git log (to view the commited files and we can move through different commited files using checkout option.master will be tha last or latest commit) git checkout commit_id (copy past the commit id by looking into log. and we can move to any commit) git checkout master (we can come back to the latest or last commit)
